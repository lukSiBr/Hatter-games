### How to apply (UPDATED FOR HATTER 2.0):
1. Download the [latest release](https://github.com/lukSiBr/Hatter-games/releases/latest)
2. Extract, select and copy all of it's files to `Hatter-main/Hatter/scalable/apps`, before running the install script. 
3. Now continue with the installation normally, open `Hatter-main/` in your terminal and run the following: 
```
./install.sh
```
